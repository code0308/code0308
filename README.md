## About Me

Hi, I’m Nahuel Gomez, a backend developer focused on designing and building scalable, high-performance server-side applications and APIs.  
I specialize in RESTful and GraphQL API development, database schema design and optimization (both SQL and NoSQL), and microservices architecture.

My core competencies include implementing secure authentication and authorization mechanisms, optimizing backend performance, and ensuring system reliability through clean, modular, and maintainable code.  
I’m proficient with technologies such as Node.js, Express, NestJS, PostgreSQL, MongoDB, Docker, Kubernetes, and various cloud platforms.

Driven by continuous learning, I actively adopt new tools and frameworks to enhance development efficiency and code quality.  
Outside of coding, I contribute to open-source projects and engage with the developer community to stay updated on the latest backend engineering trends.
